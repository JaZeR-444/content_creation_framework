## Navigation Tips

### Finding Prompts
- All prompts are numbered 001-010 within their phase folder
- Each prompt has a corresponding User-Input-Form
- Master files provide comprehensive prompt overviews

### Tracking Content
- Use `00_System/Content-ID-Map.csv` for content tracking
- Use `06_Analytics/KPI-Tracker.csv` for performance metrics
- Use `05_Distribution_and_Promotion/Distribution-Planner.csv` for cross-platform scheduling

### Quick Access
- Start: `00_System/000_START-HERE.md`
- Definitions: `GLOSSARY.md`
- Big Picture: `OVERVIEW.md`
- Phase Overviews: `[Phase]/README.md`
