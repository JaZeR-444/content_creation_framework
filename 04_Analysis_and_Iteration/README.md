# 04_Analysis_and_Iteration

This phase provides tools and processes for analyzing content performance, gathering audience insights, and continuously improving your content strategy.

## Overview

Analysis and Iteration contains 10 essential modules for data-driven content improvement:

1. **Build a Dashboard** - Track core KPIs in one place
2. **Review Retention Graphs and Drop Fix Log** - Identify and fix audience drop-off points
3. **A/B Testing Log** - Systematically test titles, thumbnails, and hooks
4. **Mine Comments Weekly** - Extract insights from audience feedback
5. **Track Competitor Watchlist** - Monitor competitor strategies and trends
6. **Set Experiment Cadence** - Establish systematic testing rhythms
7. **Prune or Refresh Underperformers** - Optimize or remove low-performing content
8. **Refresh Winners** - Extend the life of high-performing content
9. **Schedule Reviews** - Maintain weekly, monthly, and quarterly review cycles
10. **Maintain Community Health** - Foster engagement and build loyalty

## How to Use This Phase

1. **Review the Master File**: Study `Analysis-and-Iteration_Master-File.md` for the complete analytics framework
2. **Set Up Your Dashboard First**: Module 001 creates your command center for all other analyses
3. **Use Input Forms**: The `User-Input-Forms/` folder contains templates for tracking and analysis
4. **Make It Routine**: These modules work best when executed on a regular cadence

## Files in This Folder

- **Individual Prompt Files** (001-010): Detailed prompts for each analysis module
- **Master Files** (.md and .txt): Combined reference documents with all 10 modules
- **User-Input-Forms/**: Structured templates for analytics and iteration

## Success Criteria

You've completed this phase when you have:
- A functioning analytics dashboard tracking key metrics
- A/B testing process with documented learnings
- Weekly comment mining workflow capturing audience insights
- Scheduled review cadences (weekly/monthly/quarterly)
- Data-driven decision making embedded in your workflow

## Continuous Improvement

This phase creates a feedback loop back to **01_Strategy_and_Planning**. Use insights from analysis to refine your:
- Content pillars and ideal viewer profile
- Publishing cadence and content calendar
- Optimization and distribution strategies
- Production workflows and templates

## Next Steps

Analysis and Iteration is an ongoing process. Use insights gained here to continuously refine all previous phases and improve content performance over time.
