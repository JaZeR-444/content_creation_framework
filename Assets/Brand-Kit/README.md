# Brand-Kit

## Purpose

This folder contains your brand identity materials and guidelines. It's your single source of truth for visual branding across all content and platforms.

## What to Store Here

### Logo Files
- Primary logo (full color, various formats)
- Secondary/alternate logos
- Logo variations (horizontal, vertical, icon-only)
- Light and dark versions
- Transparent backgrounds (PNG)
- Vector files (SVG, AI, EPS)

### Brand Guidelines
- Brand style guide PDF
- Color palette specifications (HEX, RGB, CMYK)
- Typography guidelines
- Logo usage rules
- Spacing and clear space requirements
- What NOT to do with your brand

### Templates
- Thumbnail templates
- Social media graphics templates
- Presentation templates
- Email signature templates

### Brand Assets
- Brand patterns or textures
- Icon sets
- Graphic elements
- Photography style examples

## File Naming Convention

- `Logo_Primary_Full-Color.png`
- `Logo_Secondary_White.svg`
- `Logo_Icon-Only_Black.png`
- `Brand-Guidelines_2025.pdf`
- `Color-Palette_Specifications.pdf`

## Color Palette Example Structure

Document your colors clearly:
```
Primary Colors:
- Brand Blue: #1E40AF
- Brand White: #FFFFFF

Secondary Colors:
- Accent Orange: #F97316
- Accent Green: #10B981

Neutral Colors:
- Dark Gray: #1F2937
- Light Gray: #F3F4F6
```

## Typography Example

Document your fonts:
```
Primary Font: Montserrat
- Headings: Montserrat Bold
- Subheadings: Montserrat SemiBold
- Body: Montserrat Regular

Secondary Font: Open Sans
- Alternative body text
```

## Best Practices

1. **Keep master files** - Never delete original, editable versions
2. **Export multiple formats** - Have PNG, SVG, JPG, and AI/PSD versions
3. **Document everything** - Write down all specifications
4. **Version control** - Update brand guidelines as your brand evolves
5. **Share widely** - Make sure everyone on your team has access
