# Fonts

## Purpose

This folder stores all typography files used in your content, including brand fonts, title fonts, and any custom typefaces.

## What to Store Here

- Font files (TTF, OTF, WOFF, WOFF2)
- Font licenses and documentation
- Typography pairing guides
- Custom fonts
- Web font files
- Font specimen sheets

## File Naming Convention

Use the font family and variant name:
- `Montserrat-Bold.ttf`
- `Montserrat-Regular.ttf`
- `OpenSans-SemiBold.otf`
- `CustomBrandFont-License.txt`

## Font Organization

### By Category
Create subfolders if you have many fonts:
- Fonts/Headings/
- Fonts/Body/
- Fonts/Display/
- Fonts/Monospace/

### Font Pairing Document

Create a simple text file documenting your font combinations:
```
Font-Pairing-Guide.txt

Primary Pairing:
- Headings: Montserrat Bold
- Body: Open Sans Regular
- Works well for: Professional, clean content

Secondary Pairing:
- Headings: Playfair Display
- Body: Lato
- Works well for: Elegant, sophisticated content
```

## License Tracking

Always store font licenses with the font files:
- `Montserrat-License.txt`
- `CommercialFontName-License.pdf`
- `FontPurchaseReceipt.pdf`

**Document:**
- Where font was obtained
- License type (personal, commercial, etc.)
- Usage restrictions
- Attribution requirements (if any)

## Best Practices

1. **Verify licenses** - Ensure you have rights for commercial use
2. **Keep font files organized** - Don't mix personal and brand fonts
3. **Install system-wide** - Install fonts on your editing computer
4. **Backup font files** - Include in your backup strategy
5. **Document usage** - Note which fonts are used where
6. **Limit font count** - Stick to 2-3 font families for consistency

## Common Font Formats

- **TTF (TrueType Font):** Universal, works everywhere
- **OTF (OpenType Font):** More features, high quality
- **WOFF/WOFF2 (Web Open Font Format):** For web use
- **EOT (Embedded OpenType):** Legacy web format

## Usage Guidelines

### For Video Thumbnails
- Bold, thick fonts work best
- Ensure readability at small sizes
- Test on mobile devices
- Maintain brand consistency

### For Video Text Overlays
- Clear, legible fonts
- Good contrast with background
- Appropriate size for screen
- Limited animation for readability

### For Graphics and Social
- Platform-appropriate sizing
- Consistent with brand guidelines
- Readable on mobile
- Accessible color contrast

## Installation Tips

### Windows
1. Right-click font file
2. Select "Install" or "Install for all users"

### Mac
1. Double-click font file
2. Click "Install Font" in Font Book

### For Video Editors (Premiere, Final Cut, DaVinci)
- Install fonts system-wide first
- Restart editing software to recognize new fonts
- Keep font files accessible for project portability
