## License Documentation

**Critical:** Always track music licenses and attribution requirements.

### Create a Music-License-Tracker.csv:
```
Track-Name,Source,License-Type,Attribution-Required,Commercial-Use,Expiration-Date,License-File,Notes
Upbeat-Intro,AudioJungle,Royalty-Free,No,Yes,None,License-12345.pdf,Purchased 2025-01-15
Chill-Background,YouTube Audio Library,Free,No,Yes,None,N/A,Downloaded 2025-02-20
```

### License Types to Know

1. **Royalty-Free**
   - One-time purchase, use forever
   - May have attribution requirements
   - Verify commercial use rights

2. **Creative Commons**
   - Often free
   - Check specific CC license (BY, SA, NC, ND)
   - Attribution usually required
   - May not allow commercial use (NC restriction)

3. **Public Domain**
   - Completely free to use
   - No attribution needed
   - Confirm actual public domain status

4. **Subscription Services**
   - Valid while subscription is active
   - Check license after cancellation
   - Examples: Epidemic Sound, Artlist, Soundstripe

5. **YouTube Audio Library**
   - Free for YouTube videos
   - Check if allowed on other platforms
   - Some require attribution

## File Formats

### For Editing (Highest Quality)
- WAV (uncompressed, large files)
- AIFF (uncompressed, Mac-friendly)
- FLAC (lossless compression)

### For Final Export
- MP3 (compressed, good quality, smaller files)
- AAC (compressed, used in video containers)

### Bit Rate Recommendations
- Music: 256 kbps or higher
- Voiceover: 128-192 kbps
- Sound effects: 192-256 kbps

## Best Practices

1. **Always Check Licenses**
   - Read terms before using
   - Save license documentation
   - Understand attribution requirements
   - Verify platform permissions (YouTube vs. other platforms)

2. **Organize Consistently**
   - Use clear folder structure
   - Tag files with keywords
   - Create playlists or collections for easy access

3. **Quality Control**
   - Use high-quality source files
   - Normalize audio levels
   - Remove silence at start/end
   - Ensure seamless loops if needed

4. **Backup**
   - Include in your backup strategy
   - Keep original downloads
   - Save license files with tracks

## Attribution Best Practices

When attribution is required, include in:
- Video description
- Credits section of video
- Website or show notes
- Legal-and-Compliance/Attribution-and-Licenses.md

**Attribution Format Example:**
```
Music: "Track Name" by Artist Name
Source: [Platform Name]
License: [License Type]
Link: [URL if required]
```

## Usage Guidelines

### Background Music
- Keep volume low (don't overpower dialogue)
- Choose mood-appropriate tracks
- Ensure consistent style across content
- Use fade in/out for smooth transitions

### Intro/Outro
- Keep intros short (5-10 seconds max)
- Make it memorable and branded
- Use consistently for brand recognition
- Consider creating custom audio logo

### Sound Effects
- Use sparingly for emphasis
- Ensure quality matches video production
- Don't overuse (can become annoying)
- Match sound to on-screen action

## Legal Compliance

Track all music usage for compliance:
1. Document every track used
2. Save all licenses and receipts
3. Note attribution requirements
4. Track subscription status
5. Monitor license expiration dates
6. Reference Legal-and-Compliance/Attribution-and-Licenses.md

## Audio Editing Tips

- Normalize audio levels across tracks
- Use EQ to blend music with dialogue
- Apply compression for consistent volume
- Fade in/out to avoid abrupt starts/stops
- Duck music under dialogue (lower volume when speaking)
