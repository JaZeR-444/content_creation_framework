# Assets

## Overview

The Assets folder is your centralized media library for all content creation materials. This folder organizes visual, audio, and brand assets used across your content production workflow.

## Folder Structure

```
Assets/
├── B-Roll/              ← Background footage and supplemental video
├── Brand-Kit/           ← Logos, colors, brand guidelines
├── Fonts/               ← Typography files
├── Music/               ← Audio tracks and sound effects
├── Raw-Captures/        ← Unedited source footage
├── Stock/               ← Stock photos, videos, graphics
└── Thumbnails/          ← Thumbnail designs and templates
```

## What's Inside

### B-Roll/
Background footage and supplemental video clips that enhance your main content.

**Contents:**
- Generic B-roll footage
- Location shots
- Product shots
- Screen recordings
- Transition clips
- Overlay elements

### Brand-Kit/
Your brand identity materials and guidelines.

**Contents:**
- Logo files (PNG, SVG, AI)
- Color palette specifications
- Brand guideline documents
- Typography guides
- Usage examples
- Brand templates

### Fonts/
Typography files used in your content.

**Contents:**
- Font files (TTF, OTF, WOFF)
- Font licenses
- Typography pairing guides
- Custom fonts
- Web fonts

### Music/
Audio tracks and sound effects.

**Contents:**
- Background music tracks
- Intro/outro music
- Sound effects
- Audio stings
- Music licenses
- Attribution files

### Raw-Captures/
Unedited source footage from recordings.

**Contents:**
- Raw video files from shoots
- Multiple takes
- Audio recordings
- Screen captures
- Behind-the-scenes footage
- Backup copies

### Stock/
Stock media from third-party sources.

**Contents:**
- Stock photos
- Stock video clips
- Stock graphics
- Icons and illustrations
- Purchased assets
- License documentation

### Thumbnails/
Thumbnail designs and templates.

**Contents:**
- Thumbnail templates (PSD, AI, Canva)
- Exported thumbnails (PNG, JPG)
- A/B test variants
- Archive of published thumbnails
- Thumbnail elements (text, shapes, photos)

## How to Use This Folder

### Organizing Media

1. **Save everything in the appropriate subfolder**
   - Keep strict folder discipline
   - Use consistent naming (see `00_System/Conventions.md`)
   - Don't create duplicate copies

2. **Name files descriptively**
   - Include date: `YYYY-MM-DD`
   - Include Content-ID when applicable: `CCF-001_Thumbnail_A.png`
   - Use hyphens for spaces: `Product-Demo-B-Roll_2025-11-07.mp4`

3. **Maintain licenses and attribution**
   - Save license files with assets
   - Document attribution requirements
   - Track usage rights and expiration dates

4. **Archive old assets**
   - Create dated archive subfolders
   - Keep published assets for reference
   - Remove unused assets periodically

## Best Practices

### Storage and Backup

1. **Use Cloud Storage**
   - Google Drive, Dropbox, or similar
   - Automatic backup and sync
   - Accessible from anywhere

2. **Follow 3-2-1 Backup Rule**
   - 3 copies of important files
   - 2 different storage types
   - 1 offsite backup

3. **Organize by Project When Needed**
   - Create Content-ID subfolders for large projects
   - Keep related assets together
   - Link back to primary folders

### File Management

1. **Delete Unused Assets**
   - Review quarterly
   - Archive before deleting
   - Free up storage space

2. **Optimize File Sizes**
   - Compress large video files
   - Use appropriate formats
   - Balance quality and file size

3. **Version Control**
   - Save iterative versions
   - Use clear version naming (v1, v2, final)
   - Don't overwrite originals

### Legal and Licensing

1. **Track Licenses**
   - Save license files with assets
   - Note usage restrictions
   - Document attribution requirements
   - See `Legal-and-Compliance/Attribution-and-Licenses.md`

2. **Respect Copyright**
   - Only use properly licensed assets
   - Give credit where required
   - Track license expiration dates

3. **Commercial Use**
   - Verify commercial usage rights
   - Keep proof of purchase/license
   - Understand attribution requirements

## Recommended File Formats

### Video
- **Editing:** ProRes, DNxHD (high quality)
- **Final Export:** MP4 (H.264), MOV
- **Archive:** Original camera format

### Audio
- **Editing:** WAV, AIFF (uncompressed)
- **Final Export:** MP3, AAC
- **Archive:** WAV

### Images
- **Editing:** PSD, AI, SVG (editable)
- **Final Export:** PNG (transparency), JPG (photos)
- **Thumbnails:** PNG or JPG, 1280x720px

### Graphics
- **Vector:** SVG, AI, EPS
- **Raster:** PNG (with transparency), JPG

## Storage Size Estimates

Plan your storage needs:
- **Raw footage:** 10-50 GB per video
- **B-Roll library:** 50-500 GB
- **Stock assets:** 10-100 GB
- **Thumbnails:** 1-5 GB
- **Music library:** 5-20 GB
- **Brand kit:** 1-5 GB

**Recommended minimum:** 500 GB to 1 TB

## Troubleshooting

**Problem:** Can't find a specific asset
**Solution:** Use descriptive file names, create an asset inventory CSV, use search tools

**Problem:** Running out of storage
**Solution:** Archive old projects, compress large files, upgrade storage plan

**Problem:** Lost track of licenses
**Solution:** Create a license tracker in Legal-and-Compliance/, save license files with assets

**Problem:** Files are disorganized
**Solution:** Set aside time for organization, follow naming conventions strictly, delete unused assets
