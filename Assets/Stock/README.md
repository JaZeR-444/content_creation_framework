# Stock

## Purpose

This folder stores stock media from third-party sources, including stock photos, videos, graphics, icons, and illustrations used to enhance your content.

## What to Store Here

- Stock photos
- Stock video clips
- Stock graphics and illustrations
- Icon sets
- Vector graphics
- Purchased assets from stock sites
- License documentation
- Asset attribution records

## License Types

### 1. Free Stock (No Attribution)
- **Unsplash** - Free, no attribution required
- **Pexels** - Free photos and videos
- **Pixabay** - Free images and videos
- **Videvo** - Free video clips

**Rights:** Usually free for commercial use, no attribution needed (but always verify)

### 2. Free Stock (Attribution Required)
- **Freepik** (free tier)
- **Flaticon** (free tier)
- **Vecteezy** (free tier)

**Rights:** Free but require attribution in video description or credits

### 3. Royalty-Free (Purchased)
- **Adobe Stock**
- **Shutterstock**
- **iStock**
- **Getty Images**

**Rights:** One-time purchase, commercial use allowed, no attribution needed

### 4. Subscription Services
- **Adobe Stock** (with Creative Cloud)
- **Envato Elements**
- **Storyblocks**
- **Shutterstock Subscription**

**Rights:** Valid while subscription active, check post-cancellation rights

### 5. Creative Commons
- **Flickr** (some images)
- **Wikimedia Commons**
- **Creative Commons Search**

**Rights:** Varies by specific CC license (BY, SA, NC, ND) - always check

## Understanding Creative Commons Licenses

- **CC BY:** Attribution required
- **CC BY-SA:** Attribution + share-alike
- **CC BY-NC:** Attribution + non-commercial only
- **CC BY-ND:** Attribution + no derivatives
- **CC0:** Public domain, no restrictions

**For commercial content:** Only use CC BY, CC BY-SA, or CC0 (avoid NC restriction)

## Best Practices

### Before Download

1. **Verify License**
   - Read terms carefully
   - Check commercial use permissions
   - Note attribution requirements
   - Understand platform restrictions

2. **Check Quality**
   - Download highest resolution available
   - Verify file format is compatible
   - Preview before download

3. **Document Everything**
   - Save license files
   - Screenshot license page
   - Save download confirmation

### After Download

1. **Rename Immediately**
   - Use consistent naming convention
   - Include license or source info
   - Make files searchable

2. **Save License**
   - Keep license PDF with file
   - Update Stock-License-Tracker.csv
   - Note in Legal-and-Compliance/Attribution-and-Licenses.md

3. **Organize**
   - Place in appropriate subfolder
   - Tag with keywords
   - Link to Content-ID if used

## Attribution Requirements

### When Attribution is Required

Include in:
1. Video description
2. Credits section of video (if applicable)
3. Blog post or article
4. Legal-and-Compliance/Attribution-and-Licenses.md

### Attribution Format

**For Photos:**
```
Photo by [Photographer Name] from [Source]
License: [License Type]
Link: [URL if required]
```

**For Icons:**
```
Icons made by [Creator] from www.flaticon.com
```

**For Freepik:**
```
Image by [Creator] on Freepik
Link: [URL]
```

## File Format Guide

### Photos
- **JPG:** Photography, complex images
- **PNG:** Graphics with transparency
- **TIFF:** Highest quality, large files
- **WebP:** Modern format, smaller file size

### Videos
- **MP4:** Most common, widely compatible
- **MOV:** High quality, larger files
- **AVI:** Legacy format
- **WebM:** Web-optimized

### Graphics/Vectors
- **SVG:** Scalable vector graphics
- **AI:** Adobe Illustrator files
- **EPS:** Vector format, widely compatible
- **PNG:** Rasterized graphics

### Icons
- **PNG:** Raster icons, specific sizes
- **SVG:** Scalable icons, best choice
- **IconFont:** Font-based icons

## Quality Guidelines

### Photos
- Minimum: 1920x1080 (Full HD)
- Preferred: 3840x2160 (4K)
- For thumbnails: 1280x720 minimum

### Videos
- Minimum: 1920x1080 @ 30fps
- Preferred: 3840x2160 @ 60fps
- Match your primary content quality

### Graphics
- Vector formats when possible (SVG, AI)
- If raster, use 300 DPI for print
- Use transparent backgrounds (PNG) when appropriate

## Troubleshooting

**Problem:** Can't find license information
**Solution:** Download license file at time of download, screenshot license page, contact support

**Problem:** Uncertain about attribution
**Solution:** When in doubt, attribute; include in video description

**Problem:** Need to prove license ownership
**Solution:** Keep all license files, email confirmations, and purchase receipts

**Problem:** Subscription expired, unsure about assets
**Solution:** Review subscription terms, don't use assets if license expired, re-purchase if needed
