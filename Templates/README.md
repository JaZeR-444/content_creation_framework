# Templates

## Overview

This folder contains reusable templates to streamline your content creation workflow. These templates provide consistent starting points for common content tasks.

## What's Inside

### 1. Content-Brief.md
Template for planning and briefing content pieces before production.

### 2. Platform-Metadata.csv
Spreadsheet template for managing metadata (titles, descriptions, tags) across multiple platforms.

### 3. Script-Template.md
Structured template for writing video or podcast scripts.

### 4. Short-Form-Checklist.md
Quick checklist for creating short-form content (TikTok, Reels, Shorts, etc.).

### 5. README.md (This File)
Documentation for the Templates folder.

## How to Use These Templates

### Method 1: Copy and Customize
1. Open the template file
2. "Save As" with your Content-ID and title
3. Fill in all sections
4. Save in appropriate project folder

### Method 2: Use as Reference
1. Keep template open while working
2. Ensure you cover all template sections
3. Adapt to your specific needs

### Method 3: Create Your Own Versions
1. Customize templates for your niche
2. Add or remove sections
3. Save personalized versions
4. Share with team members

## Best Practices

1. **Keep Templates Simple**
   - Easy to understand
   - Quick to fill out
   - Not overwhelming

2. **Update Regularly**
   - Refine based on experience
   - Remove unused sections
   - Add helpful prompts

3. **Include Examples**
   - Show filled-in examples
   - Demonstrate best practices
   - Provide inspiration

4. **Make Them Accessible**
   - Store in cloud for team access
   - Easy to find and open
   - Clear naming conventions

5. **Don't Over-Template**
   - Allow for creativity and flexibility
   - Templates guide, not constrain
   - Adapt as needed per project

## Troubleshooting

**Problem:** Templates feel restrictive
**Solution:** Treat as guides, not rules; customize freely

**Problem:** Templates too long to fill out
**Solution:** Simplify, remove unnecessary sections, create "quick" versions

**Problem:** Team not using templates
**Solution:** Show benefits, make easily accessible, gather feedback

**Problem:** Templates don't fit workflow
**Solution:** Customize for your process, create new templates from scratch
